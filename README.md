# Insurance_Analytics
An interactive Branch Performance Dashboard built using Power BI to track branch performance across New Business, Cross-Sell, and Renewals. It compares targets vs achievements, analyzes the sales pipeline, and highlights key KPIs to support data-driven decision-making.

**Key Points**
1.Tracks branch performance across New Business, Cross-Sell, and Renewals
2.Compares targets vs actual achievements to measure success
3.rovides sales pipeline analysis (opportunities, meetings, deal stages)
4.Identifies high and low performing branches/individuals
5.Displays key KPIs like revenue, win ratio, conversion rate, achievement %
6.Uses interactive visualizations for better insights and decision-making
7.Built using Power BI, Excel, and DAX for data modeling and analysis
8.Enables data-driven decision-making with a centralized dashboard view
